编程语言与计算机四大件其实是密切相关的；

在学完任何一门语言的基本特性之后(变量、逻辑控制语句、面向对象思想)，接下来都需要进入这四个阶段：

  • 熟悉该语言常用的数据结构

    比如C++的vector、set、Map

    Java的List、Set、Map

    Python的tuple、字典…

    这些都是一致的

    LeetCode刷题很重要！！！

  • 熟悉该语言的多线程、多协程开发

    比如C++、Java、Python的多线程开发;

    在开发过程中，都会遇到操作系统中锁、线程并发的思想。

    要求高一点，就是高并发了。

  • 熟悉计算机网络

    互联网的核心就是通信设备之间互联，靠的就是互联网。

    C++、Java、Python都有类似Socket编程的东西，有的语言直接给封装好了。

    现在流行的框架也都是基于Socket延伸的

  • 熟悉数据库

    JDBC 这个在各个语言中都是有的。

    再要求高一点，就是数据库优化之类的了。

在熟悉了这些东西之后

就是要求从源码的层面来了解了。

提到看源码，主要问的就是实现细节，以及用到的设计模式了
